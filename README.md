# AstrlGlass

AstrlGlass is a dark glass Discord theme by AstrlZoom, based on the MIT-licensed `midnight-discord` theme.

## Install

Download `themes/AstrlGlass Theme.theme.css` and place it in your Vencord or BetterDiscord themes folder.

Direct download:

```text
https://raw.githubusercontent.com/AstrlZoom/AstrlsThemes/main/themes/AstrlGlass%20Theme.theme.css
```

Enable `AstrlGlass Theme.theme.css` in Discord, then press `Ctrl+R` if Discord does not refresh right away.

## Customize

Most visual edits are in `themes/AstrlGlass Theme.theme.css`:

- Colors: `--astrl-*`, `--accent-*`, `--bg-*`, `--text-*`
- Spacing: `--gap`, `--divider-thickness`, `--border-thickness`
- Roundness: `--radius-sm`, `--radius-md`, `--radius-lg`
- Transparency/glass: `--desktop-tint`, `--bg-floating`, `--blur-amount`, `--liquid-glass`, `--liquid-glass-*`
- Window controls: `--custom-window-controls`, `--window-control-*`
- Background image: `--background-image`, `--background-image-url`
- DMs icon: `--dms-icon-svg-url`, `--dms-icon-svg-size`, `--dms-icon-color-*`

`build/AstrlGlass.css` and `dist/AstrlGlass Theme.theme.css` are generated files. Edit the theme/source files instead.
