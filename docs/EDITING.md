# AstrlGlass Editing Guide

This project is set up so most changes happen in one file:

`themes/AstrlGlass Theme.theme.css`

Start there when you want to change colors, roundness, spacing, transparency, background images, or the visible theme metadata.

## What To Edit

- `themes/AstrlGlass Theme.theme.css`: easy visual settings, theme name, colors, spacing, roundness, and on/off switches.
- `src/dms-button.css`: custom DMs/home icon shape, fixed animation, selected-state fixes.
- `src/desktop-transparency.css`: see-through Discord window support.
- `scripts/theme.config.js`: file order for the build. Only touch this when adding or removing CSS files.

## What Not To Edit

- `build/AstrlGlass.css`: generated by the build script.
- `dist/AstrlGlass Theme.theme.css`: generated single-file theme.
- `node_modules/`: installed packages.

If you edit generated files by hand, your changes can disappear the next time the watcher rebuilds.

The project name and filenames are AstrlGlass now.

## Visual Settings

The safest beginner edits are color, spacing, roundness, transparency, and background values in `themes/AstrlGlass Theme.theme.css`.

## Publishing

Because the repo is public, other people can install the online `themes/AstrlGlass Theme.theme.css` import.

The generated `dist/AstrlGlass Theme.theme.css` file can still be installed manually because it contains everything in one file.
